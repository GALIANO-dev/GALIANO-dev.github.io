Descripción del Proyecto

Este proyecto permite digitalizar las operaciones de un restaurante mediante el uso de tablets para pedidos, pizarras digitales para cocina y emisión de boletas vía red inalámbrica. Está desarrollado bajo la metodología Scrum, gestionado en GitHub y con automatización CI/CD.

Funciones Clave

Menú Interactivo

El cliente escanea un código QR y accede al menú digital.

Pedidos a Cocina

Los pedidos se envían automáticamente a la pizarra de cocina.

Facturación Inalámbrica

El camarero emite boletas desde una tablet conectada a red.
